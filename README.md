### 1 Задание
### 1) Постановка задачи

> Ваша задача – посмотреть, какие значения принимает последовательность при разных a0 и n, вывести закономерность, по которой строится последовательность и запрограммировать ее самостоятельно.
### 2) Алгоритм

#### Алгоритм выполнения программы:

1. **Ввод данных:**  
   Программа считывает целое число и количество членов, которые нужно вывести, обозначенные как `first`,`n`.

2. **Цикл длиной n:**  
    Последовательность строится на алгоритме, который вычитает из четных чисел 3 и умножает нечетные на 4.

3. **Вывод результата:**  
   На экран выводится последовательность чисел длины n, где числа последовательности - любое целое число.

```

### 3) Программа

```java

import java.io.PrintStream;
import java.util.Scanner;
public class Main {
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;
    public static void main(String[] args) {
        int first = in.nextInt();
        int n = in.nextInt();
        for (int i=0; i<n;i++) {
            if (i%2==0) {
                first-=3;
            } else {
                first*=4;
            }
            out.print(first+" ");
        }
    }
}



```

### 2 Задание
### 1) Постановка задачи

> Дана последовательность из n целых чисел. Определите, можно ли переставить ее элементы так, чтобы остатки от деления на 3 у любых двух соседних элементов различались. Выведите "YES" или "NO".
### 2) Алгоритм

#### Алгоритм выполнения программы:

1. **Ввод данных:**  
   Программа считывает целое число n, переменную для массива mas, а затем n целых чисел - элементов последовательности.

2. **Подсчет чисел по остаткам:**  
   Программа подсчитывает количество чисел с остатком от деления на 3 равным 0 (обозначены переменной c0). То же самое проделывает для остатков 1 и 2 (обозначены переменными c1 и c2 соответственно).

3. **Вывод результата:**  
   Если число всего одно, то на экран выводится `YES`, если разница между количеством чисел с остатками 1 и 2 меньше или равна количеству чисел с остатком 0 +1, количество чисел с остатками 1 и 2 больше 0 и количество чисел с остатком 0 меньше или равна количеству чисел с остатками 1 и 2 +1, то тоже выводится `YES`, иначе выводится `NO`.
```

### 3) Программа

```java

import java.io.PrintStream;
import java.util.Scanner;
public class Main {
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;
    public static void main(String[] args) {
        int n = in.nextInt();
        int[] mas = new int[n];
        for (int i=0;i<n;i++) {
            mas[i] = in.nextInt();
        }
        int c0=0;
        int c1=0;
        int c2=0;
        for (int i=0;i<n;i++) {
            if (mas[i]%3==0) {
                c0+=1;
            } else if (mas[i]%3==1) {
                c1+=1;
            } else {
                c2+=1;
            }
        }
        if (n==1) {
            out.print("YES");
        } else if ((Math.abs(c1-c2)<=c0+1&&(c1>0||c2>0))&&c0<=c1+c2+1) {
            out.print("YES");
        } else {
            out.print("NO");
        }
    }
}


```

### 3 Задание
### 1) Постановка задачи

> На вход подается целое число M (1 ≤ M ≤ 10^15). Выведите наибольшее целое k, для которого выполняется неравенство k * k < M.

### 2) Алгоритм

#### Алгоритм выполнения программы:

1. **Ввод данных:**  
   Программа считывает целое число M.

2. **Находим корень**  
   Находим квадратный корень из числа M через функцию Math.sqrt, затем присваиваем k его значение
   
3. **Проверка на случай неточности**  
   При переводе корня из double в long его значение округляется вниз и становится правильным ответом. Однако, если корень получился целым, то k*k получится равным M, а потому от k нужно отнять единицу.
   
4. **Вывод результата:**  
   На экран выводится результат `k`.

```

### 3) Программа

```java


import java.io.PrintStream;
import java.util.Scanner;
public class Main {
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;
    public static void main(String[] args) {
        long m = in.nextLong();
        long k = (long)Math.sqrt(m);
        if (k*k==m) {
            k--;
        }
        out.println(k);
    }
}



```

### 4 Задание
### 1) Постановка задачи
> Дан массив из n элементов. Проверить, есть ли три элемента, расстояние между любыми двумя из которых не превышает k, а значения совпадают. Выведите само значение и индексы элементов, если такие элементы существуют, или "NO", если их нет.

### 2) Алгоритм

#### Алгоритм выполнения программы:

1. **Ввод данных:**  
   Программа считывает натуральное число n, натуральное число k, переменную для массива list, переменную для проверки существования элементов check, n целых чисел - элементов массива.

2. **Тройной цикл**  
    Программа через цикл и два вложенных цикла проверяет все тройки элементов на равенство, а так же на то, что расстояние между ними не превышает k.

3. **Вывод результата:**  
   Если тройка найдена, то на экран выводится значение ее элементов, а затем их индексы. Если тройку найти не удалось, то выводится `NO`.

```

### 3) Программа

```java

import java.io.PrintStream;
import java.util.Scanner;
public class Main {
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;
    public static void main(String[] args) {
        int n = in.nextInt();
        int k = in.nextInt();
        int[] list = new int[n];
        boolean check = false;
        for (int z=0; z<n;z++) {
            list[z]=in.nextInt();
        }
        for (int i=0;i<n;i++) {
            for (int x=i+1;x<n;x++) {
                if (list[x]==list[i]&&x-i<=k) {
                    for (int y=x+1;y<n;y++) {
                        if (list[y]==list[i]&&y-x<=k&&y-i<=k&&check==false) {
                            out.println(list[i]);
                            out.println(i+" "+x+" "+y);

                            check=true;
                        }
                    }
                }
            }
        }
        if (check==false) {
            out.println("NO");
        }
    }
}



```
